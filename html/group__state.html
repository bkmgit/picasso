void ps_save ( ps_context *  ctx)

Pushes a copy of the current graphics state on to stack for context.

  • Gamma value
  • Alpha value
  • Blur value
  • Text stroke color
  • Text fill color
  • Compositing method
  • Transform matrix
  • Line width
  • Line cap
  • Line join
  • Line dash
  • Stroke object
  • Source object
  • Fill color
  • Fill rule
  • Clip region
